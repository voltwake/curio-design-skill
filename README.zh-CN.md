**Curio** 是一座精选设计风格库——包豪斯、Stripe、无印良品、孟菲斯、蒸汽波、北斋浮世绘、装饰艺术，以及另外约 500 套。每一套都是完整的 token 化设计系统（色彩、字体、间距、阴影、组件），**并且**附带文化考据和可直接使用的 CSS 技法。

**本 skill** 给你的 AI agent 提供把任何一套 Curio 设计应用到任何 HTML 形态所需的一切——演示文稿、官网落地页、海报、报告、社交卡片、邮件、Dashboard，或者任何你能用 HTML 渲染的东西。你提需求，AI 装裱。

## 安装

### Claude Code

```bash
git clone https://github.com/voltwake/curio-design-skill.git ~/.claude/skills/curio
```

之后在任何 Claude Code session 里说：`"用 Curio 包豪斯做一份 10 页 pitch deck"` —— skill 会自动被 `curio` / `用 Curio` / 主题名 / 输出形态等触发词激活。

### Cursor / Codex / Gemini / OpenCode / Qwen / Aider

clone 到你的 AI agent 能读到的任何目录,然后把 agent 的 "skills" 或 "rules" 路径指过去。顶层的 `SKILL.md` 是权威入口；`CLAUDE.md` 和 `AGENTS.md` 是 1 行的 `@SKILL.md` redirect,兼容不同平台的约定。

### 不安装也能用

[designbycurio.com](https://designbycurio.com) 的每一条分享链接都返回完整的 `DESIGN.md`。把它喂给任何 AI agent + 一句话"请用这份设计做一份 {deck|落地页|海报|carousel|报告|邮件}"就够了——不过 AI 如果同时能看到本 skill 的 `SKILL.md`(里面有所有输出都必须遵守的通用规则),产出质量会更高。

## 工作原理

skill 故意做得很简单：

```
   ┌─────────────────────────────────────┐
   │  designs/<id>.md                    │   ← 视觉语言
   │                                     │     (色彩 · 字体 · 间距
   │  tokens + 文化背景 +                │     · 阴影 · 组件 · 等等)
   │  CSS 技法 + Do's & Don'ts           │
   └────────────────┬────────────────────┘
                    │
                    │  +
                    ▼
   ┌─────────────────────────────────────┐
   │  SKILL.md                           │   ← 怎么读一份 DESIGN.md
   │                                     │     + 通用设计准则
   │  阅读指南 + 通用规则 + 工作流       │     (布局 · 疏密 · 色彩
   │                                     │     · 层级 · 个性 · 防御)
   └────────────────┬────────────────────┘
                    │
                    ▼
              漂亮的 HTML 产物
          (你要什么形态都行)
```

**本 skill 不规定你做什么形态。** 你要 deck 它做 deck,你要 newsletter 它做 newsletter,你要邮件模板它做邮件模板。形态由你的需求决定;视觉语言由所选 design 决定。

## 几个调用例子

```
"用 Curio 包豪斯给我做一份 10 页的 pitch deck,气候科技初创公司"
→ 读 designs/bauhaus-weimar.md + SKILL.md
→ 产出 HTML deck —— 红蓝黄三原色,Josefin Sans 小写标题,
  硬偏移阴影,0 圆角,主色块分节页,奶油纸底色

"用孟菲斯风格做一份关于 5 条创业经验的小红书 carousel"
→ 读 designs/memphis-sottsass-1981.md + SKILL.md
→ 产出 N 张卡片 (封面 + 5 张要点 + 收尾),1080×1440 画布,
  涂鸦边框,糖果色,不对称布局

"用 Stripe 2024 风格做一份产品发布的邮件 newsletter"
→ 读 designs/stripe-2024.md + SKILL.md
→ 产出 HTML 邮件 —— 靛蓝点缀,呼吸感留白,自信稳重的字体排版。
  不需要任何 "method" —— AI 自己组合形态。

"用包豪斯做一份设计 talk 的海报"
→ 读 designs/bauhaus-weimar.md + SKILL.md
→ AI 可以瞥一眼 examples/poster.md 看看构图参考,
  然后产出一份真正的包豪斯海报 (几何网格 · 三原色 · 康定斯基色形符号)
```

## Curio 哲学

> *「不必从头起笔,取一件奇珍,余下让 AI 装裱。」*

Curio 的核心信念:好设计应当是一座**库**,而不是一项**服务**。每一个产出过强视觉语言的文化时刻——包豪斯、装饰艺术、浮世绘、蒸汽波、从 Stripe 到 Discord 的品牌系统——都应当**可携带**:任何人都能用,任何地方都能装,零学习曲线。

如果你曾经想做一份"包豪斯 deck"或一个"无印良品风的落地页",但因为自己不是设计师而做不出来——Curio 填的就是这个缺口。
